# Marketing Site

Animated marketing landing page with hero section, testimonials, and pricing tiers.

**Stack:** Astro | **Complexity:** simple
**Tags:** `marketing`, `landing`, `astro`, `tailwind`, `testimonials`, `pricing`

## Quick Start

```bash
npm install
npm run dev
```

## Docker

```bash
docker compose up --build
```
